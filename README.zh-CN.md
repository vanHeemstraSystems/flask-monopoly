# 烧瓶垄断

基于“如何使用 Flask 蓝图和 Flask-SQLAlchemy 构建大型 Flask 应用程序”，位于<https://www.digitalocean.com/community/tutorials/how-to-structure-a-large-flask-application-with-flask-blueprints-and-flask-sqlalchemy>

基于“Flask SQLAlchemy”，位于<https://github.com/vanHeemstraSystems/flask-sqlalchemy/>

基于“工厂模式”<https://github.com/vanHeemstraSystems/factory-pattern>

基于“烧瓶垄断”<https://github.com/KrzysztofWelc/flask-monopoly>

按如下方式运行该应用程序：

1) 输入`flask_app`目录：`$ cd flask_app`2) ~运行`pip install -r requirements.txt`~ 将 app/sample.env 复制到 app/.env：`$ cp app/sample.env app/.env`3）更新`app/.env` with your configuration setting.
4) If non-existent, create a virtual environment inside the `flask_app`目录：`python3 -m venv .venv`5）启动虚拟环境，输入：`. .venv/bin/activate`6）将Flask App设置到app目录：`(.venv) $ export FLASK_APP=app`7）设置Flask环境为开发：`(.venv) $ export FLASK_ENV=development`8）运行烧瓶应用程序：`(.venv) $ flask run`9）根据提示打开Web界面
10) 使用`CTRL+c`退出网络服务器。
11) 或者运行flask命令行界面：`(.venv) $ flask shell`12）执行任何flask命令：>>>
13) 使用`exit()`退出命令行界面。
